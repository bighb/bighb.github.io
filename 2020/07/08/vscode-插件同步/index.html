发表于 2020-07-08

vscode插件同步工具—settings-sync

settings-sync简介

settings-sync是一款vscode插件，是一款将你本地的插件进行云同步，当你换了一台电脑，只需要进行下载它，进行云更新就可以获取之前备份的插件，非常的方便

下载云端插件：shift+alt+d

上传本地插件：shift+alt+u

使用方法

  1. 到vscode插件库下载settings-sync插件
  2. 在vscode中按快捷键shift+ctrl+p打开设置面板，搜索sync高级选项，如图：
  3. 选择”打开设置”，如图：

4.填写相应的GIST ID 和 令牌（token)

GIST ID 获取方式

  1. 登录github

  2. 选择Your gists

  3. 进入gists网页，点击新建，如图：

  4. 填写相关描述，如图：

  5. 点击create secret gist之后，可以看到网页地址上面的GIST ID,如图：

    令牌(token)

    1. 到github，选择Settings
    2. 选择Developer settings,如图：

3.选择Personal access tokens

4.如果已经有token则直接点击，进入之后点击Regenerate token进行查看token,如图：

5.此时已经获取到token

6.如果先前没有创建，则选择Generater new token进行创建,如图：

最后讲获取到的GIST ID和TOKEN填写到vscode中的“打开设置”中，使用快捷键就可以完成插件的上传下载！

gist.github.com无法访问

  1. windows下 找到C:\Windows\System32\drivers\etc\hosts文件

  2. hosts文件 拉到桌面,用记事本打开,在最后行添加 http://ping.chinaz.com/gist.github.com，获取一个最快的区域ip

  3. 新建一个hosts文件，进行替换C盘，C盘里面不能直接修改保存hosts
