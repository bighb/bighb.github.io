在windows安装sass的坑

发表于 2020-07-08 | 分类于 sass

安装sass中间遇到的挫折

sass属于css众多预处理器中的一种，它基于ruby语言开发而成，所以在安装sass之前先来安装一下ruby

安装ruby

我的电脑是window10,安装ruby

  • ruby官网

    下载完ruby，直接安装，直接下一步，此时安装完了ruby，具体步骤不做多概述，网上有很多介绍
    安装完成后需测试安装有没有成功,运行CMD输入以下命令：
    ruby -v
    //如安装成功会打印
    ruby 2.2.2p95 (2015-04-13 revision 50295) [i386-mingw32]

    安装sass

    如上已经安装成功。但因为国内网络的问题导致gem源间歇性中断因此我们需要更换gem源。
    1.删除原gem源
    gem sources –remove https://rubygems.org/
    2.增加国内镜像（淘宝镜像已不再支持）已有下面这个地址所托管
    gem sources -a https://gems.ruby-china.org/
    3.查看是否已经增加国内镜像
    gem sources -l
    CURRENT SOURCES
    https://gems.ruby-china.org/
    此时已经讲gem源准备好
    4.再安装sass,如下命令(如mac安装遇到权限问题需加 sudo gem install sass)
    gem install sass
    5.安装完成之后，你应该通过运行下面的命令来确认应用已经正确地安装到了电脑中：
    sass -v
    Sass 3.x.x (Selective Steve)
    6.如下sass常用更新、查看版本、sass命令帮助等命令：
    //更新sass
    gem update sass

//查看sass版本
sass -v

//查看sass帮助
sass -h

此时sass和ruby全部安装完成！
